# Soil data, otu table as data frame.

A data frame made from a soil \`phyloseq\` object with only otu count
data.

## Usage

``` r
soil_phylo_otu
```

## Format

A phyloseq-class experiment-level object with an OTU table.

- otu_table:

  OTU table with 7,770 taxa and 119 samples
