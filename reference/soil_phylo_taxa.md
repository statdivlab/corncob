# Soil data, taxa table as data frame.

A data frame made from a soil \`phyloseq\` object with only taxonomy
data.

## Usage

``` r
soil_phylo_taxa
```

## Format

A phyloseq-class experiment-level object with an OTU table.

- tax_table:

  taxonomy table
