# Small soil phylum data for contrasts examples, otu table as data frame

A small subset of
[`soil_phylo_otu`](https://statdivlab.github.io/corncob/reference/soil_phylo_otu.md)
used for examples of testing contrasts. A data frame made from the
\`phyloseq\` object with only otu counts.

## Usage

``` r
soil_phylum_contrasts_otu
```

## Format

A phyloseq-class experiment-level object with an OTU table.

- otu_table:

  OTU table with 39 taxa and 56 samples
